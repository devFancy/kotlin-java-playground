# Kotlin-Java-Playground

Kotlin, Java 기반으로 다양한 실험과 연습 코드를 담은 저장소입니다.

언어와 기술을 배우는 과정에서 기록하고 싶은 부분을 정리하고 있습니다.

## Skills

> Backend
* Kotlin (1.6/1.9), Java (8/11/17)
* Spring Boot 2.7.x, 3.2.x, Spring MVC, Spring Data JPA(Hibernate)
* Kotest, Mockk, Junit5, Mockito
* Gradle (7.x, 8.x)
* RestDocs, Swagger

> DevOps
* Linux: CentOS 7.x/8.x, Ubuntu 18.04/20.04
* AWS: EC2, RDS, S3
* MySQL (5.7/8.x)
* Docker, Shell Script
* Github Action
* Nginx

> Collaboration & Tools
* Collaboration: Jira, Confluence, Slack, Notion
* Development Tools: IntelliJ, VS Code, Git, SourceTree
